Setup Required

The app is not connected to Google Sheets yet. Follow these steps:

  1. Open Google Drive → create a new Google Sheet.
  2. In the Sheet, click Extensions → Apps Script.
  3. Delete any existing code and paste the contents of Code.gs.
  4. Click Deploy → New Deployment → Web App.
    Set Execute as: Me  |  Who has access: Anyone.
  5. Copy the Web App URL.
  6. Open config.js in this project folder and replace
    PASTE_YOUR_APPS_SCRIPT_URL_HERE with the URL.
  7. Reload this page.
